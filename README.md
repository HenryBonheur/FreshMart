# FreshMart Billing 

# Project Name
FreshMart Billing Calculator

# Description
FreshMart Billing Calculator is a simple web-based point-of-sale (POS) system that allows users to add grocery items, calculate totals, and determine change after payment.  

It is designed using **HTML, CSS, and JavaScript** and simulates a real-world supermarket billing system with a clean and interactive interface.

# Features
- Add items to cart (Bread, Milk, Rice, Sugar, Oil)
- Auto-filled price based on selected item
- Quantity-based total calculation
- Live cart table display
- Automatic total calculation
- Change calculator after payment
- Clear cart functionality
- Responsive and simple UI

# Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

# How It Works
1. Select an item from the dropdown
2. Enter quantity
3. Click **Add Item**
4. View items in cart table
5. Enter amount paid
6. Click **Calculate Change**


# Project Structure
- `home.html` → Main structure
- `fresh.css` → Styling
- `app.js` → Functionality
